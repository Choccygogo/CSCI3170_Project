# CSCI3170_Project

#### • Use mySQL_JDBC.jar that we provide and Compile and run your Java program:

##### Compile:

```bash
javac -cp .:lib/mysql-jdbc.jar -d bin src/*.java
```

##### Run:

```bash
java -cp bin:lib/mysql-jdbc.jar Main
```

#### • Importing required packages for JDBC API

import java.sql.\*;
